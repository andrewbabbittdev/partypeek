## What is Party Peek?
Party Peek is a plugin for peeking at who's in a party before joining.

## What is in this repo?
This repository contains all source code for Party Peek. Everything needed to develop or deploy Party Peek is self contained in this repository.

## Getting started
To start developing Party Peek you need just a few things:

- [.NET 10 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/10.0)
- [Final Fantasy XIV](https://na.finalfantasyxiv.com/)
- [Dalamud](https://dalamud.dev/faq/getting-started/)

## Run PartyPeek
Once you have all the dependencies listed above load the plugin in Dalamud and the plugin will automatically show party members when opening a party finder listing.
